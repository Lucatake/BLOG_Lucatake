    O que é o SLnova?

    O SLnova (StarLogo Nova) é um software e uma linguagem de programação baseada em agentes. Ele é um projeto do MIT (Instituto de Tecnologia de Massachusets) que busca criar nos estudantes habilidades de investigação científica. O site do SLnova pode ser acessado clicando aqui.

    Linguagem de Programação Orientada por Agentes

    Nesta linguagem de programação, temos agentes e, cada um deles busca implementar uma ação, realizar uma tarefa específica. Os agentes trabalham sobre o paradigma da orientação a objetos e, juntos, formam e agregam ao programa final.

    Aprendendo SLnova

    O site do Project GUTS apresenta diversos materiais e tutorias para aprender a utilizá-lo. Uma ferramenta que pode ser utilizada para fins pessoais e para fins educativos.

    Salve a Floresta!

      Instruções:
    • Nesse jogo você é um cachorro e deve coletar os fogos, andando com as setas, afim de salvar a floresta de morrer.
    • Quando conseguir salvar a floresta, os Leões voltam e a floresta volta a ter seu ciclo sozinha, com as presas (Pássaros) e os predadores (Leões).

    Programa híbrido: Biology Simulation + First-Person Action Game, usando o Game of Life como inspiração. Foi feito por Luana Catake para um projeto de faculdade em Outubro 2020.
